# M5Stack-ESPHome-YAML
YAML for various M5Stack microcontrollers and addons for use with ESPHome in Home Assistant.

## NanoC6_Unit_Relay.yaml
[M5Stack NanoC6](https://docs.m5stack.com/en/core/M5NanoC6) Controller with [Unit Relay](https://docs.m5stack.com/en/unit/relay)

Working: 
- Unit Relay (GPIO via grove interface)
- Blue status LED
- RGB power and control
- CPU temp
- BLE tracker and proxy
- Uptime
- Hardware RNG
- NanoC6 button

Not working: IR transmitter
