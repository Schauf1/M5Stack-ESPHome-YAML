# M5Stack-ESPHome-YAML
YAML for various M5Stack microcontrollers and addons for use with ESPHome in Home Assistant.

NanoC6_Unit_Relay
M5Stack NanoC6 Controller with Unit Relay
Working:
-Unit Relay
-Status LED
-RGB power and control
-CPU Temp
-BLE Tracker and Proxy
-Uptime
-Hardware RNG
-NanoC6 Button
